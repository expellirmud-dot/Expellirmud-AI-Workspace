# Expellirmud AI-Workspace

Central workspace for coordinating AI-assisted work across multiple
applications, models, and product projects.

Use this directory as the primary Codex project:

```text
D:\ai-tools\AI-Workspace
```

## Modules

```text
AI-Workspace/
├─ ai-ops-registry/   Central registry, project profiles, templates, and reports
└─ README.md
```

Product repositories remain outside this workspace. For example:

```text
D:\lumina-studio
```

## Current Mode

Version 1 is manual-safe. The workspace prepares context, task cards, dispatch
instructions, and reports. It does not auto-send prompts, invoke subagents,
trigger CI pipelines, or edit product repositories without an approved task.

## Start Here

1. Read `AGENTS.md`.
2. Read `WORKSPACE.md`.
3. Open `workspace-modules.yaml`.
4. Enter `ai-ops-registry/` when registry details are needed.
